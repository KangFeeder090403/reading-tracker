# 📚 reading-tracker

Selamat datang di **reading-tracker**!  
Aplikasi ini bertujuan membantu Anda memantau dan mengelola buku atau bacaan yang telah, sedang, dan ingin Anda baca.

---

## ✨ Fitur Utama

- **Tambah Buku** — Simpan daftar buku yang ingin atau sedang Anda baca.
- **Update Progress** — Catat kemajuan bacaan setiap saat.
- **Riwayat Bacaan** — Lihat buku-buku yang sudah selesai dibaca.
- **Statistik Bacaan** — Tampilkan jumlah buku yang dibaca dan statistik lainnya.
- **Cari & Filter** — Temukan buku dengan mudah berdasarkan judul, penulis, atau status.

---

## 🚀 Cara Menggunakan

1. **Clone repo ini:**
   ```bash
   git clone https://github.com/KangFeeder090403/reading-tracker.git
   cd reading-tracker
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Jalankan aplikasi:**
   ```bash
   npm start
   ```
